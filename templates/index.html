ResNet $500

ResNet is a convolutional neural network (CNN) architecture that has significantly deeper networks compared to previous models. It is widely used for image classification tasks and has achieved state-of-the-art performance on various benchmark datasets such as ImageNet.

VGG16 $600

VGG16 is a deep convolutional neural network architecture known for its simplicity and effectiveness. It consists of 16 convolutional layers and is widely used in image recognition tasks, such as object detection and localization, due to its superior performance.

MobileNet $400

MobileNet is a lightweight deep neural network architecture designed for mobile and embedded vision applications. It is optimized for low-latency inference and small model size while maintaining high accuracy, making it suitable for various real-time image analysis tasks on mobile devices.

C3D $800

C3D (Convolutional 3D) is a deep learning architecture specifically designed for video analysis tasks. It extends convolutional neural networks (CNNs) into the spatio-temporal domain, enabling effective feature learning from both spatial and temporal information in videos. C3D is widely used for action recognition, video classification, and video segmentation.

LSTM $700

LSTM (Long Short-Term Memory) is a type of recurrent neural network (RNN) architecture capable of learning long-term dependencies in sequential data, such as videos. It is widely used for video captioning, video summarization, and video prediction tasks by capturing and modeling temporal dependencies effectively. LSTM has shown promising results in various video-related applications.

BERT $900

BERT (Bidirectional Encoder Representations from Transformers) is a transformer-based model developed by Google. It has revolutionized natural language processing (NLP) tasks by pretraining on large text corpora and fine-tuning for specific downstream tasks. BERT excels in tasks like sentiment analysis, question answering, named entity recognition, and text classification.

GPT-3 $1000

GPT-3 (Generative Pre-trained Transformer 3) is a state-of-the-art language model developed by OpenAI. It is trained on a massive dataset and can generate human-like text across various domains, including articles, poems, code, and more. GPT-3 is utilized for text generation, summarization, translation, and creative writing, showcasing remarkable language understanding and generation capabilities.
